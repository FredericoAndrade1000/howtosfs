Controlador remoto

O controlador remoto é uma peça utilizada para obter o controle de um foguete no Spaceflight Simulator. O controlador remoto é controlado a distância, por isso, não tem necessidade de usar astronautas seu interior.

Detalhes:

Tipo de peça Básica
Peso 3t
Torque (giroscópio) 5kN
Velocidade Crítica para dano 5.5m/s
