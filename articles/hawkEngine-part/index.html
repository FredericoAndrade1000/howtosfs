Hawk Engine

O motor Hawk Engine é uma peça básica de um foguete no Spaceflight Simulator, sua principal função é criar o impulso responsável por levantar o foguete do chão e direcioná-lo para um determinado destino. É uma de várias outras opções de motores para foguetes no jogo.

Detalhes:

Tipo de peça Motor
Peso 3.5t
Impulso (quanto peso consegue levantar) 120t
Eficiência 240Isp
Velocidade Crítica para dano 5.5m/s
